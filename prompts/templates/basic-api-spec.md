## 生成してほしい内容

以下の項目を含む詳細なAPI仕様書を生成してください：

1. **概要セクション**
   - APIの目的と概要
   - 対象ユーザー
   - 主な機能

2. **認証・認可**
   - 認証方法の詳細
   - 必要な権限
   - トークンの取得方法

3. **各エンドポイントの詳細**
   - リクエストの詳細（ヘッダー、ボディ、クエリパラメータ）
   - レスポンスの詳細（ステータスコード、レスポンスボディ）
   - エラーレスポンス
   - 使用例（curlコマンドまたはコードサンプル）

4. **共通仕様**
   - レート制限
   - ページネーション
   - フィルタリング・ソート

5. **エラーハンドリング**
   - エラーコード一覧
   - エラーメッセージの形式

6. **ベストプラクティス**
   - 推奨される使用方法
   - 注意事項

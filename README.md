# Empowered API Spec

AIエージェントを使用して、既存のOpenAPI仕様書を拡充し、より理解しやすく実用的なAPI仕様書を生成するためのリポジトリです。

## 概要

このリポジトリは、**既存のOpenAPI仕様書（YAML/JSON）を入力として**、AIエージェントを使ってより詳細で包括的なAPI仕様書を生成するためのプロンプトテンプレートを提供します。

### 特徴

- 📖 **ユースケース中心**: 単なるリファレンスではなく、実際の利用シーンに基づいた説明
- 🔗 **API間の関連性**: どのAPIをどの順番で呼び出すかを明確化
- 📚 **用語定義**: 専門用語を丁寧に説明
- 🎯 **手順重視**: API利用者が迷わないよう、具体的な手順を提示

## ディレクトリ構造

```
empowered-api-spec/
├── prompts/              # AIエージェント用のプロンプト
│   ├── templates/        # プロンプトのテンプレート
│   └── examples/         # 具体的な使用例
├── docs/                 # ドキュメント
└── README.md
```

## 使い方

### 1. OpenAPI仕様書を用意

既存のOpenAPI 3.0仕様書（YAML/JSON形式）を用意します。

### 2. プロンプトテンプレートを選択

`prompts/templates/openapi-enrichment.md` が主要なテンプレートです。

### 3. OpenAPI仕様書を埋め込む

テンプレート内の指定箇所に、OpenAPI仕様書の内容を貼り付けます。

### 4. AIエージェントへの入力

完成したプロンプトをAIエージェント（ChatGPT、Claude、Copilotなど）に入力します。

### 5. 拡充された仕様書を取得

AIが以下を含む詳細な仕様書を生成します：
- API全体概要とユースケース
- 各APIの詳細仕様（用語定義、関連API、利用手順を含む）
- 実用的なリクエスト/レスポンス例

## プロンプトの種類

### 主要テンプレート

- **OpenAPI拡充テンプレート** (`openapi-enrichment.md`): 既存のOpenAPI仕様書を入力として、ユースケース中心の詳細な仕様書を生成

### 補助テンプレート

- **基本的なAPI仕様書生成**: OpenAPI仕様がない場合に、ゼロから仕様書を作成
- **エンドポイント詳細仕様**: 特定のエンドポイントの詳細説明を生成
- **エラーハンドリング仕様**: エラー処理の標準化

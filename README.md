# Empowered API Spec

AIエージェントを使用して、強化したAPI仕様書を生成するためのリポジトリです。

## 概要

このリポジトリは、AIエージェントに渡すためのプロンプトを格納し、それを使ってより詳細で包括的なAPI仕様書を生成することを目的としています。

## ディレクトリ構造

```
empowered-api-spec/
├── prompts/              # AIエージェント用のプロンプト
│   ├── templates/        # プロンプトのテンプレート
│   └── examples/         # 具体的な使用例
├── docs/                 # ドキュメント
└── README.md
```

## 使い方

### 1. プロンプトの選択

`prompts/templates/` ディレクトリから適切なプロンプトテンプレートを選択します。

### 2. プロンプトのカスタマイズ

選択したテンプレートを自分のプロジェクトに合わせてカスタマイズします。

### 3. AIエージェントへの入力

カスタマイズしたプロンプトをAIエージェント（ChatGPT、Claude、Copilotなど）に入力し、API仕様書を生成します。

## プロンプトの種類

- **基本的なAPI仕様書生成**: シンプルなRESTful APIの仕様書を生成
- **OpenAPI仕様書生成**: OpenAPI 3.0形式の仕様書を生成
- **詳細なエンドポイント仕様**: 各エンドポイントの詳細な仕様を生成
- **エラーハンドリング仕様**: エラーレスポンスの詳細な仕様を生成
